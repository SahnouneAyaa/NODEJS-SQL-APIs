# 🧩 Node.js & SQL API Tutorial

A beginner-friendly tutorial created in collaboration with the Scientific Club at my university to teach students how to build RESTful APIs using Node.js, Express, and direct SQL integration.

---

## Overview
This project is part of a collaborative workshop series where I contributed as an instructor, helping design and deliver sessions focused on teaching students how to build backend APIs.

Each session of the workshop covered a different backend technology (MongoDB, GraphQL, Prisma, Sequelize, SQL), and this project represents the **Node.js + SQL module**.

The goal was to introduce students to working with relational databases using raw SQL queries without relying on an ORM.

---

## Features

This API demonstrates CRUD operations using a `Student` model:

### 🎓 Student Management
- Add student
- Get all students
- Get student by ID
- Update student
- Delete student

---

## Backend Concepts Covered

This project introduces core SQL and backend concepts:

- Direct SQL queries integration with Node.js
- Connecting to MySQL database
- Structuring queries for CRUD operations
- Input validation using Joi
- Handling request/response flow in Express
---

## Tech Stack

- Node.js  
- Express.js
